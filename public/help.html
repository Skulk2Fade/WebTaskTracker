Additional Features

This application exposes several features via the API that are not directly available in the interface.

Recurring Tasks

You can create tasks that repeat automatically. When calling POST /api/tasks, include a repeatInterval value of daily, weekly, monthly, weekday, last_day or custom. Completed tasks will automatically schedule their next occurrence. For custom recurrences, provide a recurrenceRule in iCalendar RRULE format.

Attachments

Files can be attached to tasks or comments by using the attachment endpoints. Base64 data can be sent directly in the request body, or raw files can be uploaded when an ATTACHMENT_DIR is configured on the server.

POST /api/tasks/:taskId/attachments
POST /api/tasks/:taskId/attachments/upload
POST /api/comments/:commentId/attachments/upload
GET /api/attachments/:id

See the README or API reference for more details on these endpoints.
